# Snake Game

**Выполнил:** студент 1 курса группы ПИЖ-б-о-23-2(1) направления подготовки 09.03.04 «Программная инженерия» Северо-Кавказского Федерального Университета Чекалин Егор Юрьевич.

**Описание:** Реализация классической игры "Змейка" на C++ с использованием SFML. В игре отображается текущая длина змейки и рекорд прямо на экране.

---

## Файловая структура проекта

```
.
├── assets
│   └── LiberationSans-Regular.ttf   # Шрифт для отображения текста
├── CMakeLists.txt                  # Файл сборки CMake
├── lib
│   ├── Apple.cpp                    # Реализация класса яблока
│   ├── Apple.hpp                    # Заголовок класса яблока
│   ├── GameObject.hpp               # Базовый абстрактный класс для игровых объектов
│   ├── Snake.cpp                    # Реализация класса змейки
│   └── Snake.hpp                    # Заголовок класса змейки
├── main.cpp                         # Точка входа в приложение
└── README.md                        # Этот файл

```

---

## Требования

- C++20
- SFML 3.x
- CMake 3.16+
- Linux/Arch (или другая ОС с установленной SFML)

---

## Сборка и запуск

1. Создаём папку сборки и переходим в неё:

```bash
mkdir build
cd build
````

2. Генерируем Makefile с помощью CMake:

```bash
cmake ..
```

3. Собираем проект:

```bash
cmake --build .
```

4. Запускаем игру:

```bash
./SnakeGame
```

**Примечание:** Шрифт `LiberationSans-Regular.ttf` должен находиться в папке `assets` рядом с исходниками. При использовании CMake, желательно настроить копирование `assets` в папку сборки, чтобы программа могла его найти.

---

## Управление

* **Стрелки:** движение змейки (вверх, вниз, влево, вправо)
* **Escape:** выход из игры

---

## Особенности

* Рекорд и текущая длина змейки отображаются в верхней части экрана.
* Змейка телепортируется с одного края экрана на другой.
* После столкновения с собой змейка сбрасывается, яблоко перемещается в новую случайную позицию.
